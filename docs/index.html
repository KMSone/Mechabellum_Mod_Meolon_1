Bonelab VR Input Viewer Instructions

  • Download the input viewer from here
  • The downloaded input viewer file can be opened in a browser or directly in OBS by adding a "browser" source and selecting the file
  • The input viewer should automatically connect to the SpeedrunTimer mod running on a game on the same computer (must be SpeedrunTimer v1.4.0 or higher)
  • If you are on a different computer (or using Quest) then you can connect over the network by adding ?host=x.x.x.x to the end of the browser URL where x.x.x.x is the IP address of your Quest/machine running the game

Bonelab VR Input Viewer Instructions

For PC:

  • Open the input viewer in your browser
    • If recording with OBS you should instead add it to your scene directly by adding a "Browser" source and using this for the URL: https://jakzo.github.io/SlzSpeedrunTools/input-viewer
  • The input viewer should automatically connect to the SpeedrunTimer mod running on a game on the same computer (must be SpeedrunTimer v1.4.0 or higher)

For Quest or if viewing inputs from another PC:

  • Download the input viewer from here
    • Downloading the file is required due to browser security restrictions
  • The downloaded input viewer file can be opened in a browser
    • If recording with OBS you can add it to your scene directly by adding a "Browser" source, ticking "Local file" and selecting the downloaded file
  • Press enter to open the IP address text input
    • If using OBS browser source, select the source then press the "Interact" button first
  • Write the IP address of your Quest/machine running the game then press enter again
    • To find your IP address on Quest: go to Wifi, select your network, scroll down to the advanced section and it will show your IP address
    • You can record the inputs separately and combine with the Quest recorded gameplay using video editing software later but may prefer to record both the inputs and gameplay at the same time by casting the Quest screen to the computer showing the inputs and overlaying with OBS
  • The input viewer should connect to the SpeedrunTimer mod running at the entered IP address (must be SpeedrunTimer v1.4.0 or higher)

OBS tips:

  • You can use a color key filter in OBS to make the black background transparent

Browser warning that this site is insecure?

The tools on this site require using the unencrypted HTTP protocol (rather than the encrypted and secure HTTPS) due to browser security mechanisms not allowing these tools to function correctly. Specifically an HTTPS page cannot initiate a connection to an HTTP server, which is required in the case of the input viewer or Livesplit connecting to the non-HTTPS server created within the game to provide the inputs/splits.

There should be no security risk for you, but here is what you should know to make up your own mind:

  • Because the connection to the site is unencrypted, any data sent between your machine and the server can be viewed by a "man in the middle" (your ISP, a hacker that has compromised a router, etc.)
  • You should never send sensitive information (password, credit card number, etc.) to the server over an insecure connection just in case someone is listening
  • There are no features on this website for sending sensitive information to the server
  • Things like inputs and splits sent from your game to the page, although not encrypted, go directly from the game to the page running in your browser, meaning they are not sent to the server and will not go through the internet (and they are not sensitive information anyway)
  • If you're technically inclined you can view the source code on Github, verify that the mod you downloaded was published by the action in this repo, inspect the site with dev tools, etc. for reassurance it is not doing anything malicious

VR Input Viewer Instructions

  • Open the input viewer in your browser
    • If recording with OBS you should instead add it to your scene directly by adding a "Browser" source and using this for the URL: http://vr.jf.id.au/input-viewer/
    • You can also instead download the input viewer by clicking here in case you cannot use the web version for whatever reason
  • The input viewer should automatically connect to the SpeedrunTimer mod running on a game on the same computer (must be SpeedrunTimer v1.4.0 or higher)
  • If viewing inputs from another PC (or Quest):
    • Press enter to open the IP address text input
      • If using OBS browser source, select the source then press the "Interact" button first
    • Write the IP address of your machine running the game then press enter again
      • To find your IP address on Quest: go to Wifi, select your network, scroll down to the advanced section and it will show your IP address
      • You can record the inputs separately and combine with the Quest recorded gameplay using video editing software later but may prefer to record both the inputs and gameplay at the same time by casting the Quest screen to the computer showing the inputs and overlaying with OBS

OBS tips:

  • To make the black background of the input viewer transparent in OBS: select the input viewer, click the "Filters" button, add a "Luma key" effect filter
  • The viewer looks best when it is square (width and height are the same)
  • You can use the mouse + shift/control to move the input viewer camera (but the default view should be fine for most people)
